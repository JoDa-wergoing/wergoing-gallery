# weRgoing Gallery

**weRgoing Gallery** is a WordPress plugin that displays photo albums
directly from **STRATO HiDrive**.
Images are loaded from external storage and presented in a modern,
responsive gallery using **PhotoSwipe v5**.

No media uploads to WordPress, no duplication, and no unnecessary storage usage.

---

## What is weRgoing Gallery?

weRgoing Gallery connects WordPress to your HiDrive account using secure OAuth authentication.
Photo albums are read directly from HiDrive folders and rendered as responsive
galleries inside WordPress pages or posts.

This makes weRgoing Gallery ideal for websites that want to keep media files
outside WordPress while still offering a fast and user-friendly gallery experience.

---

## Key features

- Display photo albums directly from STRATO HiDrive
- No media stored or duplicated in WordPress
- Modern PhotoSwipe v5 lightbox (mobile & retina optimized)
- Responsive image loading with smart resizing
- Secure OAuth authentication
- Rate limiting on the proxy endpoint
- Admin notices for connection status
- Clean integration with WordPress pages and posts

---

## Why use external storage for WordPress galleries?

- Keep WordPress backups small and fast
- Avoid media library clutter
- Reuse the same photo albums across multiple websites
- Better separation between content and storage

weRgoing Gallery is specifically designed for users of **STRATO HiDrive** who want a reliable and maintainable gallery solution for WordPress.

---

## Shortcodes

Display a single album:
```
[higallery album="/Photos/Vacation"]
```

Display all albums as clickable cards:
```
[higallery_all_albums]
```

A Gutenberg block is also available in the block editor.
